# AI Diploma Master 🎓🤖

Welcome to my **AI Diploma Master** repository! This repository contains all the code, assignments, and projects I have worked on during my Artificial Intelligence and Machine Learning Diploma. It serves as a comprehensive portfolio of my journey from basic programming to building machine learning models.

## 📂 Repository Structure

The repository is organized into several main sections based on the learning path:

### 1. Python (`/Python`)
Contains basic to advanced Python programming concepts:
- **Basics:** Variables, Loops, Functions.
- **Advanced:** Object-Oriented Programming (OOP).
- **Final Project:** Python final evaluation project.

### 2. Data Analysis & Visualization (`/Data_Analysis`)
Contains data manipulation and visualization scripts and notebooks:
- **Libraries used:** `NumPy`, `Pandas`, `Matplotlib`, `Seaborn`, and `Plotly`.
- **Dashboards:** Introduction to building interactive dashboards using `Dash`.
- **Final Project:** **Ford GoBike System Data Analysis**, including exploratory data analysis (EDA) and a final dashboard.

### 3. Machine Learning (`/ML`)
Contains algorithms implementations and ML projects:
- **Algorithms Covered:** Linear Regression, Logistic Regression, Decision Trees, K-Nearest Neighbors (KNN), and Support Vector Machines (SVM).
- **Projects:** - **Project 1: Boston Housing Prices Prediction** - A regression model to predict the value of houses in Boston using Decision Trees and Grid Search.

## 🛠️ Technologies & Tools Used
- **Programming Language:** Python 3.x
- **Data Manipulation:** NumPy, Pandas
- **Data Visualization:** Matplotlib, Seaborn, Plotly, Dash
- **Machine Learning:** Scikit-Learn (sklearn)
- **Environment:** Jupyter Notebook, VS Code
